================================================================================
SUGGESTED PROJECT PHASES (5-person team)
================================================================================

Phase 0 — Foundations
  Repo, branching, CI basics; agree MVP scope; Trello board with backlog.

Phase 1 — Data
  Ingest trail metadata (e.g. COTREX); define schemas; optional review corpus.

Phase 2 — RAG core
  Chunk text, embeddings, vector store, retrieval + grounded answers with citations.

Phase 3 — Conditions & safety
  Weather API integration; rules for time windows (e.g. “afternoon”) and risk signals.

Phase 4 — Product surface
  API orchestration + UI; natural language → structured filters + RAG.

Phase 5 — Cloud & demo
  Deploy on AWS; observability; demo script; documentation for reproducibility.
